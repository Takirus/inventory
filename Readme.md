# Inventory System
Laravel 11 + PostgreSQL приложение для учёта оборудования, сотрудников, программного обеспечения установленного на компьютерах сотрудников, с возможностью создания заявок сотрудниками при возникновении проблем связанных с оборудованием. Поддерживается история заявок, оборудования и статусов сотрудников.

## Используемые технологии
- PHP 8.3 / Laravel 11
- PostgreSQL 15.6
- Bootstrap 5
- Vite
